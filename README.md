# Manic Dungeon (in Godot)

ManicDungeon is a 2d game for Mac or PC, you enter a randomly generated dungeon and have to escape.
This is the Godot version.

## Included
**Currently, has a main menu:**
- Play: enter the dungeon
- Quit: exits program

**Volume controls for:**
- Master
- Music
- SFX

## Other info
- The Volume settings are stored in a config file when you leave the menu page.
- The settings are loaded when the program starts.

## Game scene
**Includes:**
- Floor tilemap
- Wall tilemap
- WallEdge tilemap
- Player Prefab
- Exit Prefab
- Fog tilemap
- Defogger
- Level# (increase chance of escape with each level)

**Done:**
